### Workflow: Knowledge Capture

You are capturing knowledge after a meeting, decision, or discovery. Follow this workflow:

1. Create notes with `create_note` for each decision, fact, or insight
2. Use `create_relation` to link related notes together
3. Link notes to relevant code with `create_relation` targeting the code graph
4. Link notes to relevant documentation with `create_relation` targeting the docs graph
5. Create skills with `create_skill` for any reusable procedures discovered
6. Create tasks with `create_task` for any action items identified
7. Link tasks to notes with `create_task_link` for traceability

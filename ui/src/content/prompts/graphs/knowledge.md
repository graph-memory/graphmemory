### Knowledge Graph

User-created notes, facts, and decisions with typed relations and cross-graph links. Mirrored to `.notes/` directory.

| Tool | Purpose |
|------|---------|
| `create_note` | Create a note with title, content, and tags |
| `update_note` | Update note title, content, or tags |
| `delete_note` | Delete a note and all its relations |
| `get_note` | Fetch a note by ID |
| `list_notes` | List notes with optional filter and tag |
| `search_notes` | Hybrid search over notes |
| `create_relation` | Create relation between notes or to other graphs |
| `delete_relation` | Delete a relation |
| `list_relations` | List all relations for a note |
| `find_linked_notes` | Find notes linked to a doc/code/file/task node |
| `add_note_attachment` | Attach a file to a note |
| `remove_note_attachment` | Remove an attachment |

### Workflow: Task Planning

You are planning and managing project work. Follow this workflow:

1. Use `list_tasks` to review current task status across all kanban columns
2. Use `search_tasks` to find tasks related to the current discussion
3. Use `recall_skills` to find established procedures for common work
4. Create new tasks with `create_task` and set appropriate priority and status
5. Use `link_task` to establish dependencies between tasks (subtask_of, blocks)
6. Use `move_task` to update task status as work progresses
7. Use `find_linked_tasks` to check what code/docs are affected by each task
8. Create knowledge notes for planning decisions with `create_note`

### Workflow: Project Management

You are managing project work. Follow this workflow:

1. Use `list_tasks` to review current task status across all columns
2. Use `search_tasks` to find tasks related to the current topic
3. Use `recall_skills` before starting complex work to find established procedures
4. Use `move_task` to update task status as work progresses
5. Link tasks to relevant code and documentation with `create_task_link`
6. Create knowledge notes for decisions and meeting outcomes
7. Use `bump_skill_usage` after applying a skill to track what procedures are most used

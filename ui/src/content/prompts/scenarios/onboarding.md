### Workflow: Onboarding

You are exploring this project for the first time. Follow this workflow:

1. Start with `search` and `search_code` to understand the overall architecture
2. Use `list_topics` and `get_toc` to browse documentation structure
3. Use `cross_references` to understand how code and docs relate
4. Use `explain_symbol` when you encounter unfamiliar symbols
5. Use `search_all_files` to find configuration and infrastructure files
6. Create knowledge notes to capture your understanding as you go
